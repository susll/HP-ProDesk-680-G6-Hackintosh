### 硬件配置

处理器：Intel Core i7-10700

芯片组：Intel Q470

内存：24G DDR4 2400 * 3

显卡：UHD630

声卡：ALC222

网卡：Intel I219LM11

> 无线网卡需自行购买，可以没有
无线网卡：BCM94360CD with NGFF Adapter

> 自带硬盘为三星PM991 不支持黑苹果 需更换

硬盘：WD_BLACK SN750 256G
